Built an XGBoost-based predictor with 95% accuracy and 0.94 ROC-AUC. Engineered features from voltage, current, and temperature data. Developed a Streamlit UI for real-time risk scoring, enabling an estimated 15% hazard reduction.

Tech Stack: Python, scikit-learn, XGBoost, Pandas, Streamlit.

Credit Card Fraud Detection Using ANN

Developed an ANN model on 284k+ transactions with extreme class imbalance. Utilized SMOTE and Random Search tuning. Achieved 99.64% accuracy with an 81.4% F1-score, balancing false positives/negatives to reduce financial risk.

Tech Stack: Python, TensorFlow/Keras, Pandas, NumPy, scikit-learn, Keras Tuner.
